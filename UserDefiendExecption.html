User-Defined Exceptions

In Java, user-defined exceptions are custom exceptions that are created by the programmer to suit their specific needs. These exceptions are created by extending the built-in Exception class or one of its subclasses. User-defined exceptions allow developers to handle exceptional cases that may not be addressed by the built-in exception classes. By defining their own exception classes, developers can provide more detailed information about the error, and they can also define how the exception should be handled. To create a user-defined exception, a new class is created that extends the Exception or RuntimeException class. The class can then define constructors, methods, and fields specific to the exception. When a user-defined exception is thrown, it is handled in the same way as any other exception in Java. The catch block for the exception must be defined, and the program can then perform any necessary actions to handle the exception. User-defined exceptions are a powerful tool for developers to create more robust and maintainable code. By using these exceptions, developers can create code that is more readable, easier to maintain, and better able to handle exceptional cases.

Example

            class InvalidAgeException extends Exception {

                public InvalidAgeException(String message) {
                  super(message);
                }
              }

              class Voter {

                public void checkAge(int age) throws InvalidAgeException {
                  if (age < 18) {
                    throw new InvalidAgeException("You are not eligible to vote!");
                  } else {
                    System.out.println("You are eligible to vote.");
                  }
                }
              }

              class Main {

                public static void main(String[] args) {
                  Voter voter = new Voter();
                  try {
                    voter.checkAge(15);
                  } catch (InvalidAgeException e) {
                    System.out.println("Exception caught: " + e.getMessage());
                  }
                }
              }
