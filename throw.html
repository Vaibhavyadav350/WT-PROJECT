Try-Catch-Throw-Finally

The Try-Catch-Throw block is similar to the Try-Catch-Finally block, but it allows the programmer to specify which exceptions should be handled by the catch block and which should be thrown up to the calling method using the throws keyword. This enables the calling method to handle the exception or pass it up the call stack until it is handled.

Example

        try {
          // code that may throw an exception
        } catch (ExceptionType1 e1) {
          // code to handle the exception of type ExceptionType1
          throw e1;
        } catch (ExceptionType2 e2) {
          // code to handle the exception of type ExceptionType2
          throw e2;
        } finally {
          // code that will be executed regardless of whether an exception is thrown or not
        }
